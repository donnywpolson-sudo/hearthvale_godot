# Generated Audit State

The JSON, prompt, report result, and transcript files here may be stale after workflow script edits. Do not treat them as current results unless they were produced by an explicitly approved fresh audit or improvement run.

Do not regenerate this folder by running `RUN_AUDIT.ps1`, Godot, smoke batches, simulations, or `codex exec` unless the user explicitly approves that runtime action.
